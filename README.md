# Ascot Landing Page

Simple vanilla HTML and JavaScript landing page scaffolded with Vite, styled with Tailwind CSS, icons by Iconoir, and animated with Motion.

## Stack

- Vite for local development and production builds
- Tailwind CSS for utility-first styling
- Motion for Framer-style animations in vanilla JavaScript
- Icons by Iconoir

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Public assets

Store static assets in `public/` when they should be served as-is from the site root.

Examples:

- `public/favicon.svg` is available at `/favicon.svg`
- `public/images/hero.jpg` is available at `/images/hero.jpg`
